    使用随机数生成器生成随机数 #

    随机数生成器是随机数引擎类模板的一个预定义实例

    #include <random>
#include <iostream>
int main(){
    std::random_device rd;
    for(size_t n {};n<8;++n){
        std::cout<<rd()<<" ";
    }
    std::cout << std::endl;
}
